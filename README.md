# foundry-effect-manager
Create and manage reusable Active Effects outside Actor and Item sheets.

## Current scope

This module targets Foundry VTT `14.359` and adds a dedicated sidebar tab for standalone Active Effect templates.

Stored effects:

- are not attached to any Actor or Item until applied
- can be organized into folders
- can be dragged onto Actor sheets, Item sheets, and Tokens
- are applied as normal embedded `ActiveEffect` documents on the drop target
